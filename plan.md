## 阶段划分（对齐文档 V0.1 → V0.3 与四阶段排期）

### 阶段 1：V0.1 战斗核心原型（当前阶段）
目标：30 秒内能体验"移动 → 自动攻击 → 主动射击 → 区域压力"的核心闭环。

- 第 1 课 伤害管线：IDamageable / Health / 阵营区分（所有战斗共用）
- 第 2 课 敌人基础：追击 AI、近战攻击、受击、死亡回收
- 第 3 课 自动索敌：TargetRegistry 目标注册表 + 最近目标查询
- 第 4 课 双武器最小版：自动武器（定时索敌开火）+ 主动武器（左键射击），共用投射物
- 第 5 课 区域危险：HazardZone 持续掉血，推动走位
- 第 6 课 简易刷怪器：定时生成、场上上限（先用克隆模板，对象池第 3 周再上）
- 工程 hygiene：现有脚本转 UTF-8、加命名空间、补 Attack 输入动作

### 阶段 2：搜刮与构筑（V0.2）
掉落权重 → 拾取交互 → 背包纯数据网格（可测试）→ 拖拽 UI → 合并升级 → 1 条邻接联动
原则：背包先写纯数据二维网格 + 单元测试，UI 只是投影。

### 阶段 3：成长与完整单局（V0.2 收尾）
经验/升级三选一 → 波次递增 → 积分结算 → 胜败条件 → Result 页 → 重开闭环

### 阶段 4：展示版打磨（V0.3）
精英/Boss → UI/音效/特效 → 对象池与性能 → 录屏与作品集材料

## 执行纪律（来自文档第 27 节）

- 优先顺序：核心手感 > 完整闭环 > 构筑深度 > 表现打磨 > 内容扩充
- 不服务"区域压力 + 背包构筑 + 双战斗系统"的功能一律进暂缓列表
- 每步产出必须可在 Unity 里实际运行验证
